Control Flow Statements

Go Control Flow Loops

Control flow statements determine how your Go program executes — which code paths are taken, how loops iterate, and when execution branches. Go keeps things simple with a small set of powerful constructs: if, for, switch, and a few supporting keywords.

If Statement

If statements execute an operation if a specified condition is true and can include else clauses for alternative operations.

if x > 5 {
    fmt.Println("x is greater than 5")
} else {
    fmt.Println("x is less than or equal to 5")
}

x := 5
if x > 10 {
    fmt.Println("x is greater than 10")
} else if x > 5 {
    fmt.Println("x is greater than 5 but not greater than 10")
} else {
    fmt.Println("x is less than or equal to 5")
}

For Statement

Go provides three main approaches to looping, all using the for keyword:

// Traditional for loop
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// for range to iterate over a slice
mySlice := []int{1, 2, 3, 4, 5}
for index, value := range mySlice {
    fmt.Println(index, value)
}

// Infinite loop
for {
    // execute operations
    break // use break to exit
}

Break and Continue

The break keyword exits a loop entirely, while continue skips the current iteration and moves to the next one.

for i := 1; i <= 10; i++ {
    if i == 5 {
        break // exit the loop
    }
    if i == 3 {
        continue // skip current iteration
    }
    fmt.Println(i)
}

Switch Statement

The switch statement evaluates an expression and executes the matching case. Unlike many other languages, Go’s switch cases do not fall through by default.

switch x {
case 1:
    fmt.Println("x equals 1")
case 2:
    fmt.Println("x equals 2")
default:
    fmt.Println("x is neither 1 nor 2")
}

While-like Loops

Go doesn’t have while and do-while constructs, but you can achieve the same behavior using for:

// While-like loop
i := 0
for i < 5 {
    fmt.Println(i)
    i++
}

// Do-while-like loop
i := 0
for {
    fmt.Println(i)
    i++
    if i == 5 {
        break
    }
}
