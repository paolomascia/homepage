Wireless Networks

Networking Wireless Wi-Fi Security

Wireless networks facilitate device communication through radio frequency signals without physical cables, offering flexibility, scalability, and mobility across enterprise and public environments.

Wireless Technologies

IEEE 802.11 (Wi-Fi) represents the most common standard, though specialized applications employ Bluetooth, Zigbee, and cellular alternatives.

Common Wireless Standards

Standard Frequency Band Max Data Rate Approx. Range Notes
802.11a 5 GHz 54 Mbps ~35 m Less interference, shorter range
802.11b 2.4 GHz 11 Mbps ~50 m Long range, more interference
802.11g 2.4 GHz 54 Mbps ~50 m Backward compatible with 802.11b
802.11n 2.4 / 5 GHz 600 Mbps ~70 m MIMO support
802.11ac 5 GHz 6.9 Gbps ~35 m MU-MIMO, wider channels
802.11ax (Wi-Fi 6) 2.4 / 5 / 6 GHz 9.6 Gbps ~70 m OFDMA, higher efficiency

Wireless Network Modes

Mode Description Use Case
Infrastructure Clients connect through an access point (AP) Home and enterprise Wi-Fi
Ad Hoc Peer-to-peer device communication Temporary networks, no AP
Mesh Multiple APs interconnected wirelessly Campus, outdoor, IoT networks
Wireless Bridge Links two wired segments over RF Building-to-building links
Repeater Extends wireless coverage Signal boosting, range extension

Wireless Frequency Bands

  • 2.4 GHz: Greater range, fewer channels, increased interference susceptibility
  • 5 GHz: Superior throughput, more non-overlapping channels, reduced range
  • 6 GHz (Wi-Fi 6E): Minimal interference, exceptional capacity, emerging adoption

Wireless Security

Strong encryption, authentication, and access control mechanisms are vital due to signals propagating beyond physical boundaries.

Wireless Security Protocols

Protocol Encryption Authentication Status
WEP RC4 stream cipher Static keys Obsolete — easily cracked
WPA TKIP (RC4) Pre-Shared Key (PSK) Legacy, insecure
WPA2 AES (CCMP) PSK / 802.1X (EAP) Current standard
WPA3 AES (GCMP) SAE / 802.1X Latest standard

Enterprise Authentication (802.1X)

Enterprise networks leverage external RADIUS servers for centralized access validation through the 802.1X protocol framework.

Best Practices for Wireless Security

  • Deploy WPA3 or WPA2-Enterprise with robust AES encryption
  • Eliminate legacy protocols (WEP, WPA, TKIP)
  • Implement RADIUS / 802.1X authentication in enterprise contexts
  • Disable SSID broadcasts for administrative purposes only
  • Separate guest and internal traffic via VLANs and firewall policies
  • Apply MAC filtering with awareness of limited effectiveness
  • Establish strong PSK (≥16 random characters) without enterprise authentication
  • Activate WIDS/WIPS for unauthorized AP detection

Rogue Access Points and Attacks

Attack Type Description Mitigation
Rogue AP Unauthorized AP on internal LAN WIDS/WIPS detection, NAC policies
Evil Twin Fake AP mimicking legitimate SSID 802.1X, certificate validation
Deauthentication Attack Deauth frame flooding WPA3, 802.11w (Protected Management Frames)
Packet Sniffing Unencrypted traffic capture Encrypted protocols (HTTPS, VPN)
Man-in-the-Middle Traffic interception via malicious AP Mutual authentication, TLS, VPNs

  authentication open

 !
 channel 36

 no shutdown

 switchport mode access

Wireless Channel and Power Planning

  • Utilize non-overlapping 2.4 GHz channels (1, 6, 11)
  • Prevent channel overlap between adjacent APs
  • Prioritize 5 GHz band for interference reduction
  • Calibrate transmit power for balanced coverage and minimal co-channel interference
  • Execute site surveys validating coverage and signal strength (RSSI > -67 dBm)

Wireless VLAN Integration

Multiple SSIDs map to distinct VLANs for traffic segmentation through trunk links between access points and switches.

Wireless Network Optimization Tips

  • Conduct periodic RF surveys identifying interference sources
  • Maintain current firmware for stability and security enhancements
  • Deploy load balancing across APs in high-density zones
  • Monitor client roaming patterns optimizing AP overlap (~15–20%)
  • Configure QoS for latency-sensitive applications (voice, video)
  • Employ centralized wireless controllers for large-scale administration

A well-designed wireless network balances performance, coverage, and security through advanced technologies like Wi-Fi 6, WPA3 encryption, and dynamic radio management.
