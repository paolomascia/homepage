Go 4 Microservices Crash Course

GoMicroservicesRESTKafkaDockerJWTConsul

Go was designed at Google to solve the real-world problems of large-scale distributed systems: slow compilation, poor concurrency support, and bloated runtimes. The result is a language that feels almost deceptively simple on the surface, yet scales effortlessly to handle millions of concurrent requests. This manual is your end-to-end companion for mastering Go in the context of microservices.

We begin from the ground up — installing the toolchain, understanding Go's static type system, and grasping the idiomatic patterns that make Go code clean and maintainable. From there we move into the language's crown jewel: goroutines and channels, lightweight concurrency primitives that let you write highly parallel code without the complexity of threads or callbacks. You will learn not only the basics but also advanced patterns like fan-in/fan-out pipelines, worker pools, and context-driven cancellation.

The middle sections tackle the day-to-day concerns of production services: structured logging with Go 1.21's log/slog, application configuration via flags, environment variables, and Viper, HTTP servers with routing middleware and graceful shutdown, and REST API clients with retry logic and exponential backoff. We then cover data persistence through both raw SQL with database/sql and the GORM ORM, including connection pooling, transactions, and relationship mapping.

The final part focuses squarely on microservices concerns: securing endpoints with JWT access and refresh tokens backed by bcrypt-hashed passwords; decoupling services with RabbitMQ (AMQP, dead-letter queues, publisher confirms) and Apache Kafka (consumer groups, offset management, exactly-once semantics); enabling dynamic service discovery through HashiCorp Consul with health checks and client-side load balancing; protecting sensitive data with AES-256-GCM symmetric encryption and HMAC integrity verification; and finally packaging everything into minimal Docker images via multi-stage builds and deploying to Kubernetes. By the end, you will have a complete, battle-tested toolkit for building, securing, and operating Go microservices in production.

Part 1 — Language Fundamentals

1. Installation & Hello World

Go is a compiled, statically-typed language designed for simplicity, high performance, and excellent concurrency. It compiles to a single binary with no runtime dependencies, making it ideal for microservices and containerized workloads.

Download & Install

  1. Download from https://golang.org/dl/ and choose the appropriate installer for your OS.
  2. Windows: run the .msi installer — it sets up PATH automatically.
  3. Linux/macOS: extract the archive and add Go to your shell profile: 
    tar -C /usr/local -xzf go1.22.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
  4. Verify: go version

Your First Go Program

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

Run it without compiling first: go run hello.go

Build a binary: go build -o hello hello.go && ./hello

Go Modules

Every modern Go project is a module. Create one with:

mkdir myapp && cd myapp
go mod init example.com/myapp  # creates go.mod
go get github.com/some/package  # adds dependencies, creates go.sum
Tip: go.mod declares the module path and dependency versions. go.sum contains cryptographic checksums for reproducible builds. Both must be committed to version control.

If / Else If / Else

For Loops

Go has a single for keyword that replaces while, do-while, and for from other languages.

// Classic C-style
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// While-style
i := 0
for i < 5 {
    i++
}

// Infinite loop
for {
    // do work
    break  // exit when done
}

// Range over slice
fruits := []string{"apple", "banana", "cherry"}
for idx, fruit := range fruits {
    fmt.Printf("%d: %s\n", idx, fruit)
}

// Range over map
scores := map[string]int{"Alice": 90, "Bob": 75}
for name, score := range scores {
    fmt.Printf("%s scored %d\n", name, score)
}

// Break and continue
for i := 1; i <= 10; i++ {
    if i == 5 { continue }   // skip 5
    if i == 8 { break }      // stop at 8
    fmt.Println(i)
}

Switch

day := "Monday"
switch day {
case "Monday", "Tuesday", "Wednesday", "Thursday", "Friday":
    fmt.Println("Weekday")
case "Saturday", "Sunday":
    fmt.Println("Weekend")
default:
    fmt.Println("Unknown")
}

// Switch without expression (acts like if-else)
score := 85
switch {
case score >= 90:
    fmt.Println("A")
case score >= 80:
    fmt.Println("B")
default:
    fmt.Println("C or below")
}

// Type switch (common with interfaces)
func describe(i interface{}) {
    switch v := i.(type) {
    case int:
        fmt.Printf("Integer: %d\n", v)
    case string:
        fmt.Printf("String: %s\n", v)
    default:
        fmt.Printf("Unknown type: %T\n", v)
    }
}

Defer

defer schedules a function call to run when the surrounding function returns. Defers stack (LIFO) and are commonly used for cleanup.

func processFile(path string) error {
    f, err := os.Open(path)
    if err != nil { return err }
    defer f.Close()  // always runs, even if we return early with an error

    // process file...
    return nil
}

4. Functions & Packages

Function Signatures

// Basic function
func greet(name string) string {
    return "Hello, " + name
}

// Multiple parameters of the same type
func add(a, b int) int {
    return a + b
}

// Multiple return values (idiomatic Go)
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

result, err := divide(10, 3)
if err != nil {
    log.Fatal(err)
}

// Named return values
func minMax(nums []int) (min, max int) {
    min, max = nums[0], nums[0]
    for _, n := range nums[1:] {
        if n < min { min = n }
        if n > max { max = n }
    }
    return  // naked return
}

// Variadic functions
func sum(nums ...int) int {
    total := 0
    for _, n := range nums { total += n }
    return total
}
fmt.Println(sum(1, 2, 3, 4))   // 10
fmt.Println(sum(nums...))       // spread a slice

First-Class Functions

// Function as variable
double := func(n int) int { return n * 2 }
fmt.Println(double(5))   // 10

// Function as parameter (higher-order functions)
func apply(nums []int, fn func(int) int) []int {
    out := make([]int, len(nums))
    for i, n := range nums {
        out[i] = fn(n)
    }
    return out
}
result := apply([]int{1, 2, 3}, double)  // [2, 4, 6]

// Closures capture surrounding scope
func counter() func() int {
    n := 0
    return func() int {
        n++
        return n
    }
}
c := counter()
fmt.Println(c(), c(), c())  // 1 2 3

Package Visibility

package myservice

// Exported (capital first letter) — accessible from other packages
func ProcessOrder(id string) error { /* ... */ }
var MaxRetries = 3

// Unexported (lowercase) — package-private
func validateID(id string) bool { /* ... */ }
var internalCache = map[string]bool{}
Tip: In Go, capitalization is the access control mechanism — there is no public/private keyword. This rule applies uniformly to functions, types, variables, and struct fields.

5. OOP: Structs, Methods & Interfaces

Go doesn't have classes or inheritance, but it achieves object-oriented design through structs (data), methods (behavior), interfaces (contracts), and embedding (composition).

Structs & Methods

type Rectangle struct {
    Width  float64
    Height float64
}

// Value receiver — operates on a copy
func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

// Pointer receiver — can modify the struct
func (r *Rectangle) Scale(factor float64) {
    r.Width *= factor
    r.Height *= factor
}

rect := Rectangle{Width: 10, Height: 5}
fmt.Println(rect.Area())   // 50
rect.Scale(2)
fmt.Println(rect.Area())   // 200

Embedding (Composition over Inheritance)

type Person struct {
    Name string
    Age  int
}

func (p Person) Introduce() string {
    return fmt.Sprintf("I'm %s, age %d", p.Name, p.Age)
}

type Employee struct {
    Person               // embedded — promotes all fields and methods
    CompanyID string
    Role      string
}

emp := Employee{
    Person:    Person{Name: "Alice", Age: 30},
    CompanyID: "CORP-001",
    Role:      "Engineer",
}
fmt.Println(emp.Name)        // promoted field
fmt.Println(emp.Introduce()) // promoted method

Interfaces

An interface is a set of method signatures. Any type that implements all the methods satisfies the interface implicitly — no explicit declaration needed.

type Storer interface {
    Save(key, value string) error
    Load(key string) (string, error)
    Delete(key string) error
}

// In-memory implementation
type MemStore struct {
    data map[string]string
}

func NewMemStore() *MemStore {
    return &MemStore{data: make(map[string]string)}
}

func (m *MemStore) Save(key, value string) error {
    m.data[key] = value
    return nil
}

func (m *MemStore) Load(key string) (string, error) {
    v, ok := m.data[key]
    if !ok { return "", fmt.Errorf("key not found: %s", key) }
    return v, nil
}

func (m *MemStore) Delete(key string) error {
    delete(m.data, key)
    return nil
}

// Function accepts any Storer — easy to test with mocks
func processItem(s Storer, key, val string) {
    if err := s.Save(key, val); err != nil {
        log.Fatal(err)
    }
}

The Empty Interface & Type Assertions

// any (alias for interface{}) accepts any value
func printAnything(v any) {
    fmt.Printf("value=%v type=%T\n", v, v)
}

// Type assertion
var i any = "hello"
s, ok := i.(string)   // safe assertion
if ok {
    fmt.Println("string length:", len(s))
}

Constructor Patterns

type Server struct {
    host    string
    port    int
    timeout time.Duration
}

func NewServer(host string, port int) *Server {
    return &Server{
        host:    host,
        port:    port,
        timeout: 30 * time.Second,
    }
}

func (s *Server) WithTimeout(d time.Duration) *Server {
    s.timeout = d
    return s   // fluent builder pattern
}

6. Error Handling

Go uses explicit error return values instead of exceptions. Every function that can fail should return an error as its last return value. This forces callers to handle failures immediately.

The error Interface

type error interface {
    Error() string
}

// Create simple errors
err := errors.New("something went wrong")
err = fmt.Errorf("user %d not found", userID)

// Pattern: return (value, error)
func getUser(id int) (*User, error) {
    if id <= 0 {
        return nil, fmt.Errorf("invalid user id: %d", id)
    }
    return &User{ID: id}, nil
}

user, err := getUser(0)
if err != nil {
    log.Printf("error: %v", err)
    return
}
fmt.Println(user.ID)

Wrapping Errors (Go 1.13+)

Use %w in fmt.Errorf to wrap errors — this preserves the original error for inspection with errors.Is and errors.As.

var ErrNotFound = errors.New("not found")

func findItem(id string) (*Item, error) {
    item, ok := store[id]
    if !ok {
        return nil, fmt.Errorf("findItem(%s): %w", id, ErrNotFound)
    }
    return item, nil
}

// Caller can test for specific errors
item, err := findItem("xyz")
if errors.Is(err, ErrNotFound) {
    // respond with 404
} else if err != nil {
    // unexpected error — log and 500
}

Custom Error Types

type ValidationError struct {
    Field   string
    Message string
}

func (e *ValidationError) Error() string {
    return fmt.Sprintf("validation failed on field '%s': %s", e.Field, e.Message)
}

func validateEmail(email string) error {
    if !strings.Contains(email, "@") {
        return &ValidationError{Field: "email", Message: "must contain @"}
    }
    return nil
}

err := validateEmail("notanemail")
var ve *ValidationError
if errors.As(err, &ve) {
    fmt.Println("Field:", ve.Field)
}

Panic & Recover

Use panic only for truly unrecoverable programmer errors. Use recover in deferred functions to contain panics (e.g., in HTTP middleware).

func safeHandler(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        defer func() {
            if rec := recover(); rec != nil {
                log.Printf("panic recovered: %v\n%s", rec, debug.Stack())
                http.Error(w, "internal server error", http.StatusInternalServerError)
            }
        }()
        next.ServeHTTP(w, r)
    })
}

Best Practices

  • Check errors immediately; return early on failure (“fail fast”).
  • Wrap errors with context using fmt.Errorf("op: %w", err).
  • Define sentinel errors (var ErrXxx = errors.New(...)) for reusable comparisons.
  • Use errors.Is to check error identity; errors.As to extract type.
  • Log once at the boundary — don't propagate the same error log up the call stack.

7. File Handling

Go's standard library provides everything needed for file I/O through the os, io, and bufio packages.

Writing Files

package main

import (
    "bufio"
    "os"
)

func main() {
    // Create (or truncate) and write
    f, err := os.Create("data.txt")
    if err != nil { panic(err) }
    defer f.Close()

    // Direct write
    f.WriteString("line 1\n")

    // Buffered write (more efficient for many small writes)
    w := bufio.NewWriter(f)
    w.WriteString("line 2\n")
    w.WriteString("line 3\n")
    w.Flush()  // must flush!

    // WriteFile convenience (creates + writes + closes)
    data := []byte("all at once\n")
    os.WriteFile("output.txt", data, 0644)
}

Reading Files

// Read entire file (small files only)
data, err := os.ReadFile("data.txt")
if err != nil { panic(err) }
fmt.Println(string(data))

// Line-by-line (large files, efficient)
f, _ := os.Open("data.txt")
defer f.Close()

scanner := bufio.NewScanner(f)
for scanner.Scan() {
    line := scanner.Text()
    fmt.Println(line)
}
if err := scanner.Err(); err != nil {
    log.Fatal(err)
}

Appending to Files

f, err := os.OpenFile("log.txt", os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0644)
if err != nil { panic(err) }
defer f.Close()
fmt.Fprintln(f, "new log entry")

File Management

os.Rename("old.txt", "new.txt")
os.Remove("new.txt")
os.MkdirAll("a/b/c", 0755)  // create nested directories

// Check if file exists
_, err := os.Stat("config.yaml")
if os.IsNotExist(err) {
    fmt.Println("file does not exist")
}

Best Practices

  • Always defer file.Close() immediately after a successful Open or Create.
  • Use bufio.Scanner for large text files to avoid loading everything into memory.
  • Prefer os.ReadFile / os.WriteFile for small, complete file operations.
  • Always check errors on scanner.Err() after a range loop.

8. Working with JSON

JSON is the lingua franca of microservices. Go's encoding/json package handles all common use cases natively.

Encoding (Struct → JSON)

package main

import (
    "encoding/json"
    "fmt"
)

type User struct {
    ID       int    `json:"id"`
    Name     string `json:"name"`
    Email    string `json:"email,omitempty"`  // omit if empty
    Password string `json:"-"`                // never include in JSON
}

func main() {
    u := User{ID: 1, Name: "Alice", Email: "alice@example.com", Password: "secret"}

    // Compact JSON
    data, _ := json.Marshal(u)
    fmt.Println(string(data))
    // {"id":1,"name":"Alice","email":"alice@example.com"}

    // Pretty-printed JSON
    pretty, _ := json.MarshalIndent(u, "", "  ")
    fmt.Println(string(pretty))
}

Decoding (JSON → Struct)

jsonStr := `{"id":2,"name":"Bob","email":"bob@example.com"}`
var u User
if err := json.Unmarshal([]byte(jsonStr), &u); err != nil {
    log.Fatal(err)
}
fmt.Printf("User: %+v\n", u)

HTTP Encode/Decode

// In an HTTP handler
func createUserHandler(w http.ResponseWriter, r *http.Request) {
    var user User
    if err := json.NewDecoder(r.Body).Decode(&user); err != nil {
        http.Error(w, "invalid JSON", http.StatusBadRequest)
        return
    }
    // process...
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(http.StatusCreated)
    json.NewEncoder(w).Encode(user)
}

Dynamic / Unknown Structure

// Unmarshal into a map when structure is unknown
raw := `{"event":"login","user_id":42,"metadata":{"ip":"1.2.3.4"}}`
var data map[string]any
_ = json.Unmarshal([]byte(raw), &data)
fmt.Println("event:", data["event"])

// Streaming large JSON arrays
decoder := json.NewDecoder(file)
decoder.Token()  // consume opening '['
for decoder.More() {
    var record Record
    decoder.Decode(&record)
    process(record)
}

Nested Structures

type Address struct {
    Street string `json:"street"`
    City   string `json:"city"`
}

type Profile struct {
    User    User    `json:"user"`

    Tags    []string `json:"tags"`
}

p := Profile{
    User:    User{ID: 1, Name: "Alice"},
    Address: Address{Street: "Via Roma 1", City: "Milan"},
    Tags:    []string{"admin", "beta-tester"},

fmt.Println(string(data))

Modern 12-Factor apps accept configuration from multiple sources. The canonical precedence is: Flags > Environment Variables > Config File > Defaults.

Command-Line Flags

var (
    port    = flag.Int("port", 8080, "HTTP port to listen on")

    verbose = flag.Bool("verbose", false, "Enable verbose logging")
)

func main() {
    flag.Parse()
    fmt.Printf("Starting on %s:%d\n", *host, *port)
}

Environment Variables

import "os"

dbURL := os.Getenv("DATABASE_URL")
if dbURL == "" {
    dbURL = "postgres://localhost:5432/app"  // default

// os.LookupEnv distinguishes "not set" from "empty"
jwtSecret, ok := os.LookupEnv("JWT_SECRET")

    log.Fatal("JWT_SECRET must be set")

Viper — Config Files + Env + Flags Combined

go get github.com/spf13/viper

Example config.yaml:

server:
  host: "0.0.0.0"
  port: 8080
log:
  level: "info"

  url: "postgres://user:pass@localhost:5432/app?sslmode=disable"
package main

import (
    "log"
    "strings"
    "github.com/spf13/viper"
)

type Config struct {
    Server   struct { Host string; Port int }
    Log      struct { Level string; JSON bool }
    Database struct { URL string }
}

func loadConfig() Config {
    // Defaults
    viper.SetDefault("server.host", "0.0.0.0")
    viper.SetDefault("server.port", 8080)
    viper.SetDefault("log.level", "info")

    // File
    viper.SetConfigName("config")
    viper.SetConfigType("yaml")
    viper.AddConfigPath(".")
    _ = viper.ReadInConfig()

    // Env vars with APP_ prefix (APP_SERVER_PORT overrides server.port)
    viper.SetEnvPrefix("APP")
    viper.SetEnvKeyReplacer(strings.NewReplacer(".", "_"))
    viper.AutomaticEnv()

    var cfg Config
    if err := viper.Unmarshal(&cfg); err != nil {
        log.Fatalf("config unmarshal: %v", err)
    }
    return cfg
}

Configuration Validation

func validate(cfg Config) error {
    if cfg.Server.Port <= 0 || cfg.Server.Port > 65535 {
        return fmt.Errorf("server.port out of range: %d", cfg.Server.Port)
    }
    if cfg.Database.URL == "" {
        return fmt.Errorf("database.url is required")
    }
    return nil
}

cfg := loadConfig()
if err := validate(cfg); err != nil {
    log.Fatalf("invalid config: %v", err)
}

10. Goroutines & Channels

Go's concurrency model is built on two primitives: goroutines (lightweight threads managed by the Go runtime) and channels (typed, synchronized communication pipes). The mantra is: "Don't communicate by sharing memory; share memory by communicating."

Goroutines

// Launch a goroutine with the 'go' keyword
go func() {
    fmt.Println("running concurrently")
}()

// Use sync.WaitGroup to wait for goroutines to finish
var wg sync.WaitGroup

for i := 0; i < 5; i++ {
    wg.Add(1)
    go func(id int) {
        defer wg.Done()
        fmt.Printf("worker %d done\n", id)
    }(i)
}

wg.Wait()  // block until all workers finish

Channels

// Unbuffered channel — sender blocks until receiver is ready
ch := make(chan string)

go func() {
    ch <- "hello"   // send
}()

msg := <-ch          // receive
fmt.Println(msg)

// Buffered channel — sender only blocks when buffer is full
buffered := make(chan int, 5)
buffered <- 1   // doesn't block
buffered <- 2
v := <-buffered // 1

Worker Pool (Fan-Out / Fan-In)

Distribute work across a fixed number of goroutines to control concurrency and resource usage.

package main

import (
    "fmt"
    "sync"
)

func worker(id int, jobs <-chan int, results chan<- int, wg *sync.WaitGroup) {
    defer wg.Done()
    for j := range jobs {
        results <- j * j  // simulate CPU work
    }
}

func main() {
    const numWorkers = 4
    jobs    := make(chan int, 100)
    results := make(chan int, 100)
    var wg sync.WaitGroup

    // Start workers
    for w := 0; w < numWorkers; w++ {
        wg.Add(1)
        go worker(w, jobs, results, &wg)
    }

    // Send jobs
    for i := 1; i <= 20; i++ {
        jobs <- i
    }
    close(jobs)

    // Close results when all workers finish
    go func() {
        wg.Wait()
        close(results)
    }()

    // Collect results
    for r := range results {
        fmt.Println(r)
    }
}

func generate(nums ...int) <-chan int {
    out := make(chan int)

        for _, n := range nums { out <- n }

    }()
    return out

    out := make(chan int)
    go func() {
        for n := range in { out <- n * n }

    }()
    return out

func filter(in <-chan int, pred func(int) bool) <-chan int {
    out := make(chan int)
    go func() {
        for n := range in {
            if pred(n) { out <- n }
        }
        close(out)
    }()
    return out
}

    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(code)
    json.NewEncoder(w).Encode(v)

func createBook(w http.ResponseWriter, r *http.Request) {

    respond(w, http.StatusOK, list)

    b, ok := books[id]
    booksMu.RUnlock()

    respond(w, http.StatusOK, b)
}

func doWithRetry(client *http.Client, req *http.Request, maxAttempts int) (*http.Response, error) {
    var lastErr error
    delay := 500 * time.Millisecond

    for attempt := 0; attempt < maxAttempts; attempt++ {
        // Clone request body on each attempt
        resp, err := client.Do(req)
        if err == nil && resp.StatusCode < 500 {
            return resp, nil
        }
        if err != nil { lastErr = err } else { lastErr = fmt.Errorf("server error: %d", resp.StatusCode) }

        time.Sleep(delay)
        delay *= 2
        if delay > 30*time.Second { delay = 30 * time.Second }
    }
    return nil, fmt.Errorf("after %d attempts: %w", maxAttempts, lastErr)
}

    defer rows.Close()

    var users []User
    for rows.Next() {
        var u User
        if err := rows.Scan(&u.ID, &u.Email, &u.Name, &u.CreatedAt); err != nil {
            return nil, err
        }
        users = append(users, u)
    }
    return users, rows.Err()
}

func transferBalance(ctx context.Context, db *sql.DB, fromID, toID int, amount float64) error {
    tx, err := db.BeginTx(ctx, &sql.TxOptions{Isolation: sql.LevelSerializable})
    if err != nil { return err }

    defer func() {
        if err != nil { tx.Rollback() }
    }()

    if _, err = tx.ExecContext(ctx,
        "UPDATE accounts SET balance = balance - ? WHERE id = ?", amount, fromID); err != nil {
        return err
    }
    if _, err = tx.ExecContext(ctx,
        "UPDATE accounts SET balance = balance + ? WHERE id = ?", amount, toID); err != nil {
        return err
    }

    return tx.Commit()
}

  • Use parseTime=true in DSN to scan DATETIME columns into time.Time.

Token Generation & Validation

package auth

import (
    "errors"
    "fmt"
    "time"

    "github.com/golang-jwt/jwt/v5"
)

var (
    accessSecret  = []byte(os.Getenv("JWT_ACCESS_SECRET"))
    refreshSecret = []byte(os.Getenv("JWT_REFRESH_SECRET"))
    accessTTL     = 15 * time.Minute
    refreshTTL    = 7 * 24 * time.Hour
)

type Claims struct {
    UserID   int    `json:"user_id"`
    Username string `json:"username"`
    Role     string `json:"role"`
    jwt.RegisteredClaims
}

func IssueAccessToken(userID int, username, role string) (string, error) {
    now := time.Now()
    claims := &Claims{
        UserID: userID, Username: username, Role: role,
        RegisteredClaims: jwt.RegisteredClaims{
            Issuer:    "my-service",
            Subject:   fmt.Sprintf("%d", userID),
            ExpiresAt: jwt.NewNumericDate(now.Add(accessTTL)),
            IssuedAt:  jwt.NewNumericDate(now),
        },
    }
    return jwt.NewWithClaims(jwt.SigningMethodHS256, claims).SignedString(accessSecret)
}

func ParseAccessToken(tokenStr string) (*Claims, error) {
    claims := &Claims{}
    token, err := jwt.ParseWithClaims(tokenStr, claims,
        func(t *jwt.Token) (any, error) {
            if t.Method != jwt.SigningMethodHS256 {
                return nil, fmt.Errorf("unexpected alg: %s", t.Header["alg"])
            }
            return accessSecret, nil
        },
        jwt.WithValidMethods([]string{"HS256"}),
    )
    if err != nil || !token.Valid {
        return nil, errors.New("invalid token")
    }
    return claims, nil
}

Authentication Middleware

type contextKey string
const userClaimsKey contextKey = "user"

func AuthMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        var tokenStr string

        // 1. Try cookie
        if c, err := r.Cookie("access_token"); err == nil {
            tokenStr = c.Value
        }
        // 2. Try Authorization header
        if tokenStr == "" {
            auth := r.Header.Get("Authorization")
            if strings.HasPrefix(auth, "Bearer ") {
                tokenStr = strings.TrimPrefix(auth, "Bearer ")
            }
        }

        if tokenStr == "" {
            http.Error(w, "unauthorized", http.StatusUnauthorized); return
        }

        claims, err := auth.ParseAccessToken(tokenStr)
        if err != nil {
            http.Error(w, "unauthorized", http.StatusUnauthorized); return
        }

        ctx := context.WithValue(r.Context(), userClaimsKey, claims)
        next.ServeHTTP(w, r.WithContext(ctx))
    })
}

// Helpers for handlers
func UserFromContext(r *http.Request) (*auth.Claims, bool) {
    c, ok := r.Context().Value(userClaimsKey).(*auth.Claims)
    return c, ok
}

Login & Refresh Flow

func signIn(w http.ResponseWriter, r *http.Request) {
    var creds struct {
        Username string `json:"username"`
        Password string `json:"password"`
    }
    json.NewDecoder(r.Body).Decode(&creds)

    user, err := db.FindByUsername(creds.Username)
    if err != nil || !auth.CheckPassword(creds.Password, user.PasswordHash) {
        http.Error(w, "invalid credentials", http.StatusUnauthorized); return
    }

    accessToken, _ := auth.IssueAccessToken(user.ID, user.Username, user.Role)
    refreshToken, _ := auth.IssueRefreshToken(user.ID, user.Username)

    // Secure HttpOnly cookies
    http.SetCookie(w, &http.Cookie{
        Name:     "access_token",
        Value:    accessToken,
        HttpOnly: true, Secure: true,
        SameSite: http.SameSiteLaxMode,
        Expires:  time.Now().Add(15 * time.Minute),
    })

    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"status": "ok"})
}

Security Best Practices

  • Store JWT secrets in environment variables / secret managers — never in code.
  • Use HttpOnly + Secure + SameSite=Lax cookies to prevent XSS and CSRF.
  • Set short expiry for access tokens (5–15 min) and longer for refresh tokens (7–30 days).
  • Validate all claims: issuer, expiry, not-before, signing method.
  • Rate-limit the sign-in endpoint to prevent brute-force attacks.
  • For multi-service setups, use RS256 (asymmetric) so verifiers don't need the signing key.

    defer stop()

    for {
        if err := runRabbit(ctx, "amqp://guest:guest@localhost:5672/"); err != nil {
            log.Printf("rabbit error: %v (reconnecting in 5s)", err)
            select {
            case <-ctx.Done(): return
            case <-time.After(5 * time.Second):
            }
        } else { return }
    }
}

func runRabbit(ctx context.Context, url string) error {
    conn, err := amqp.Dial(url)
    if err != nil { return err }
    defer conn.Close()

    ch, err := conn.Channel()
    if err != nil { return err }
    defer ch.Close()

    // Declare durable queue with dead-letter exchange
    ch.ExchangeDeclare("events.dlx", "fanout", true, false, false, false, nil)
    ch.QueueDeclare("events.dlq", true, false, false, false, nil)
    ch.QueueBind("events.dlq", "", "events.dlx", false, nil)

    q, _ := ch.QueueDeclare("events", true, false, false, false,
        amqp.Table{"x-dead-letter-exchange": "events.dlx"})

    ch.Qos(10, 0, false)  // prefetch 10 messages

    deliveries, _ := ch.Consume(q.Name, "", false, false, false, false, nil)

    for {
        select {
        case <-ctx.Done(): return nil
        case d, ok := <-deliveries:
            if !ok { return fmt.Errorf("channel closed") }
            if err := handleMessage(d.Body); err != nil {
                d.Nack(false, false)  // send to DLQ
                continue
            }
            d.Ack(false)  // acknowledge success
        }
    }
}

func handleMessage(body []byte) error {
    log.Printf("processing: %s", string(body))
    return nil
}

type Resolver struct {
    name   string
    client *api.Client
    mu     sync.RWMutex
    eps    []string
}

func NewResolver(name, consulAddr string) *Resolver {
    cfg := api.DefaultConfig()
    if consulAddr != "" { cfg.Address = consulAddr }
    c, _ := api.NewClient(cfg)
    r := &Resolver{name: name, client: c}
    r.refresh()
    go r.watch(10 * time.Second)
    return r
}

func (r *Resolver) refresh() {
    entries, _, _ := r.client.Health().Service(r.name, "", true, nil)
    r.mu.Lock()
    r.eps = nil
    for _, e := range entries {
        r.eps = append(r.eps, fmt.Sprintf("http://%s:%d", e.Service.Address, e.Service.Port))
    }
    r.mu.Unlock()
}

func (r *Resolver) watch(d time.Duration) {
    t := time.NewTicker(d)
    defer t.Stop()
    for range t.C { r.refresh() }
}

func (r *Resolver) Pick() (string, bool) {
    r.mu.RLock()
    defer r.mu.RUnlock()
    if len(r.eps) == 0 { return "", false }
    return r.eps[rand.Intn(len(r.eps))], true  // random load balancing
}

21. Symmetric Cryptography

Go's standard library provides robust cryptographic primitives. For symmetric encryption, use AES-GCM — an AEAD (Authenticated Encryption with Associated Data) mode that provides both confidentiality and integrity.

Warning: Never use ECB, CFB, or CBC without an HMAC — they lack integrity protection. Avoid DES/3DES entirely. Prefer AES-GCM or XChaCha20-Poly1305 (from golang.org/x/crypto).

AES-256-GCM Encryption

package crypto

import (
    "crypto/aes"
    "crypto/cipher"
    "crypto/rand"
    "encoding/base64"
    "errors"
    "io"
)

// Encrypt encrypts plaintext with AES-256-GCM.
// Returns base64(nonce || ciphertext+tag).
func Encrypt(key, plaintext []byte) (string, error) {
    block, err := aes.NewCipher(key) // key must be 16, 24, or 32 bytes
    if err != nil { return "", err }

    gcm, err := cipher.NewGCM(block)
    if err != nil { return "", err }

    nonce := make([]byte, gcm.NonceSize())
    if _, err := io.ReadFull(rand.Reader, nonce); err != nil {
        return "", err
    }

    ciphertext := gcm.Seal(nil, nonce, plaintext, nil)
    out := append(nonce, ciphertext...)
    return base64.StdEncoding.EncodeToString(out), nil
}

// Decrypt decrypts the output of Encrypt.
func Decrypt(key []byte, encoded string) ([]byte, error) {
    raw, err := base64.StdEncoding.DecodeString(encoded)
    if err != nil { return nil, err }

    block, err := aes.NewCipher(key)
    if err != nil { return nil, err }

    gcm, err := cipher.NewGCM(block)
    if err != nil { return nil, err }

    if len(raw) < gcm.NonceSize() {
        return nil, errors.New("ciphertext too short")
    }

    nonce, ciphertext := raw[:gcm.NonceSize()], raw[gcm.NonceSize():]
    return gcm.Open(nil, nonce, ciphertext, nil)
}
