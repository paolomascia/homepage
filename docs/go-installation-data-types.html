Go Installation and Data Types

Go Installation Data Types

Getting started with Go begins with a proper installation and a solid understanding of its type system. This guide walks you through installing Go on any platform and explores every data type the language offers — from primitive types like integers and strings to composite types like slices, maps, and pointers.

Installation

Download and Install Go

  1. Obtain the installation package from https://golang.org/dl/
  2. Select the appropriate version for your OS (32-bit or 64-bit)
  3. Windows: Execute the .msi file following the installation wizard
  4. Linux/macOS: Extract the tar.gz file to your preferred directory

Configure Environment Variables

Windows steps:

  • Access Control Panel → System → Advanced system settings → Environment Variables
  • Locate the PATH variable and select Edit
  • Insert Go installation path (e.g., C:\Go\bin)
  • Click OK

Linux/macOS steps:

  • Open .bashrc or .bash_profile
  • Add: export PATH=$PATH:/usr/local/go/bin
  • Update /usr/local/go with your extraction location

Verify Installation

Execute this terminal command:

go version

Success displays your installed Go version.

Hello World in Go

  1. Create a text file with .go extension (e.g., hello.go)
  2. Enter this code:
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
  1. Save the file
  2. Navigate to the file’s folder in terminal
  3. Compile: go build hello.go
  4. Run: ./hello

Code Explanation

  • package main: declares this as the main package (required)
  • import "fmt": imports Go’s standard formatting library
  • func main(): program entry point
  • fmt.Println(): displays output from the fmt library

Data Types in Go

Two categories exist: primitive and composite types.

Primitive Data Types

Type Description Example
bool Boolean value, true or false var isActive bool = true
int, int8, int16, int32, int64 Signed integers; int64 ranges from -2^63 to 2^63-1 var age int = 25
uint, uint8, uint16, uint32, uint64 Unsigned integers; uint64 ranges from 0 to 2^64-1 var count uint = 100
float32, float64 Single and double precision per IEEE 754 var price float64 = 19.99
complex64, complex128 Single and double precision complex numbers var c complex128 = 1 + 2i
string Unicode character sequences var name string = "John"

Composite Data Types

  • array: Fixed-length collection of identical types
  • slice: Variable-length collection of identical types
  • map: Key-value pairs with unique keys
  • struct: Mixed-type field collections
  • pointer: Memory address reference
  • interface: Method set definition

Special types: byte (alias for uint8) and rune (alias for int32).

Variable Declaration

Explicit Type Declaration

var x int
var y float64
var z string

Type Inference

a := 10               // a is of type int
b := 3.14             // b is of type float64
c := "Hello, world!"  // c is of type string

Multiple Declaration

var x, y, z int
a, b, c := 1, 2, 3

Initialization with Values

var x int = 10
y := 3.14

Important: Declared variables must be used; unused variables prevent compilation.

Strings

Strings are immutable, representing byte sequences of Unicode characters.

String Declaration

var message string = "Hello, World!"

String Length

length := len(message)

Accessing Characters

char := message[0]

String Concatenation

var str1 string = "Hello, "
var str2 string = "World!"
var message string = str1 + str2

String to Byte Array Conversion

bytes := []byte(message)

Byte Array to String Conversion

bytes := []byte{'H', 'e', 'l', 'l', 'o', ',', ' ', 'W', 'o', 'r', 'l', 'd', '!'}
message := string(bytes)

String Package Functions

import "strings"

var message string = "Hello, World!"

// Check if substring exists
if strings.Contains(message, "World") {
    // The substring "World" is present
}

// Replace substring
newMessage := strings.Replace(message, "World", "Go", 1)

Arrays

Homogeneous, static structures with fixed element count. Syntax: var array_name [length]type

Array Declaration and Initialization

// Declare integer array with length 5
var numbers [5]int

// Initialize with values
var numbers = [5]int{1, 2, 3, 4, 5}

// Let Go deduce length
numbers := [...]int{1, 2, 3, 4, 5}

Accessing Array Elements

firstNumber := numbers[0]
numbers[2] = 10
lastNumber := numbers[len(numbers)-1]

Iterating Over Arrays

// Traditional for loop
for i := 0; i < len(numbers); i++ {
    fmt.Println(numbers[i])
}

// Using range function
for _, value := range numbers {
    fmt.Println(value)
}

Slices

Dynamically sized structures representing “windows” into underlying arrays.

Slice Declaration

var s []type

// Create with make function
s := make([]type, length, capacity)

Slice Examples

package main

import "fmt"

func main() {
    // Declaration of an empty slice
    var s []int
    fmt.Println("empty slice:", s)

    // Declaration and initialization of a slice
    s1 := []int{1, 2, 3, 4, 5}
    fmt.Println("initialized slice:", s1)

    // Access slice values
    fmt.Println("s1[0]:", s1[0])
    fmt.Println("s1[1]:", s1[1])

    // Modify slice values
    s1[2] = 100
    fmt.Println("modified s1:", s1)

    // Slice length
    fmt.Println("length of s1:", len(s1))

    // Add element to slice
    s1 = append(s1, 6)
    fmt.Println("s1 with new element:", s1)

    // Add multiple elements
    s1 = append(s1, 7, 8, 9)
    fmt.Println("s1 with more elements:", s1)

    // Create slice from another slice
    s2 := s1[1:4]   // Elements from index 1 to 3
    s3 := s1[2:]    // Elements from index 2 to end
    s4 := s1[:3]    // Elements from start to index 2
}

Searching in Slices

func findInSlice(slice []int, target int) bool {
    for _, value := range slice {
        if value == target {
            return true
        }
    }
    return false
}

// Usage
numbers := []int{2, 4, 6, 8, 10}
found := findInSlice(numbers, 6)

Maps

Data structures associating keys with values, similar to dictionaries.

Map Creation and Operations

// Create empty map
myMap := make(map[string]int)

// Add key-value pairs
myMap["apple"] = 3
myMap["orange"] = 5
myMap["banana"] = 2

// Access values
fmt.Println(myMap["apple"])

// Check if key exists
_, ok := myMap["banana"]
if ok {
    fmt.Println("The key 'banana' exists in the map")
} else {
    fmt.Println("The key 'banana' does not exist in the map")
}

// Iterate over map
for key, value := range myMap {
    fmt.Printf("Key %s has value %d\n", key, value)
}

Constants Declaration

Constants are immutable values defined using the const keyword.

const myConstant = 3.14

const myBoolean = true
const myString = "Hello, world!"
const myInteger = 42
const myRune = 'A'

// Multiple constants
const (
    pi = 3.14159
    e  = 2.71828
)

Constants prevent accidental modification of values that shouldn’t change during execution.

Variable Visibility

Two types exist based on naming conventions:

Public Visibility

Uppercase-starting names are accessible across packages:

package mypackage

var MyVariable int = 10

// Usage in another package
package main

import "mypackage"

func main() {
    fmt.Println(mypackage.MyVariable)
}

Private Visibility

Lowercase-starting names are package-restricted:

package mypackage

var myVariable int = 10
// myVariable only accessible within mypackage

Pointers

Variables containing memory addresses of other variables.

Pointer Declaration and Usage

var p *int

var x int = 42
var p *int = &x

fmt.Println(*p)

// Create new value in memory
p := new(int)

// Using pointers in functions
func double(x *int) {
    *x *= 2
}

func main() {
    x := 42
    double(&x)
    fmt.Println(x)
}
